## 5. User Flows

### 5.1. Data Processing Flow
```
1. User navigates to /process
2. User uploads file (drag & drop or browse)
3. System validates file format
4. User selects extraction methods
5. User configures parameters
6. User clicks "Start Processing"
7. System creates task and shows progress
8. Real-time updates via WebSocket
9. On completion, redirect to /results
10. User views and exports results
```

### 5.2. Results Analysis Flow
```
1. User navigates to /results
2. User applies filters (date, type, location)
3. System fetches filtered data
4. User reviews data table
5. User clicks on row to view details
6. Modal shows full extraction details
7. User can edit or export specific record
8. User views analytics charts
9. User exports filtered results (CSV/JSON)
```

### 5.3. Configuration Flow
```
1. Admin navigates to /admin
2. Admin updates system settings
3. Admin saves configuration
4. System validates and applies changes
5. System restarts affected services
6. Confirmation message shown
```

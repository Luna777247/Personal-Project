# Simulation Experiments

This directory contains experiment configurations and result analysis for the climate change impact models.

## Experiment Structure

### Flood Impact Experiments
- **Baseline Scenario**: Normal rainfall conditions
- **Extreme Flood**: High intensity rainfall
- **Urban Planning**: Different drainage configurations

### Drought Impact Experiments
- **Mild Drought**: Gradual water reduction
- **Severe Drought**: Rapid water depletion
- **Irrigation Strategies**: Different water management approaches

### Heavy Rain Experiments
- **Storm Events**: Variable storm intensities
- **Infrastructure Capacity**: Testing drainage limits
- **Population Response**: Evacuation behavior analysis

## Running Experiments

1. Open model in GAMA Platform
2. Load experiment configuration
3. Set parameters
4. Run multiple replications
5. Export results for analysis

## Result Analysis

Use the provided Python scripts in the root directory to:
- Process simulation outputs
- Generate statistical summaries
- Create visualization plots
- Compare different scenarios

## Key Metrics

### Flood Model
- Flooded household percentage
- Evacuation rates
- Recovery times

### Drought Model
- Crop failure rates
- Water usage efficiency
- Farmer adaptation strategies

### Heavy Rain Model
- Flooded area coverage
- Drainage system utilization
- Resident displacement

## Parameter Sensitivity

Experiments should test:
- Climate parameter variations
- Population behavior changes
- Infrastructure modifications
- Policy interventions
